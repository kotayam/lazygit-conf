# Lazygit Config

My personal lazygit config.

---

## Installation

### 1. Clone this repo

```bash
git clone git@github.com:kotayam/lazygit-conf.git
```

### 2. Run setup.sh

```bash
cd lazygit-conf
chmod +x setup.sh
./setup.sh
```
